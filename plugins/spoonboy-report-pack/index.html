Spoon Boy Report Pack

A suite of additional reports in a single plugin designed to extend the management information available in Morpheus.

Extend your Management Information.

This report pack is built for the post v1.0 plugin framework so requires Morpheus version later than 6.3.0. New reports are being added frequently, currently the pack contains the following addtional reports:

User Reports

A suite of reports that make it simple to inspect the status of the user accounts accessing your Morpheys appliance.

  • 2FA Status Report
  • Account Locked Report
  • Account Disabled Report
  • Password Expired Report
  • Failed Login Attempts Report
  • Logged In Users Report

Provisioning Inventory Reports

Additional reports to augment your provisioning inventory reporting.

  • Morpheus Agent Installed Version Report
